# Month Info University

Модуль для консольных приложений на C#, определяющий по номеру месяца время года и учебный семестр/сессию/каникулы для вуза.

## Функциональность

- Ввод номера месяца (1-12)
- Определение времени года
- Определение учебного периода (семестр/сессия/каникулы)
- Обработка некорректного ввода

## Пример работы

Введите номер месяца (1-12):
12
Это Зима, Сессия

Введите номер месяца (1-12):
7
Это Лето, Каникулы

Введите номер месяца (1-12):
13
Это Некорректный номер месяца,

## Куда можно встроить

- В учебные расписания
- В календарные приложения для студентов
- В системы планирования учебного процесса
- В информационные киоски вузов

## Требования

- .NET Core / .NET Framework
- Консольное приложение

## Примечание

Логика соответствует учебному году в российских вузах: 1 семестр (сентябрь-декабрь), сессия (январь), 2 семестр (февраль-май), сессия (июнь), каникулы (июль-август).
